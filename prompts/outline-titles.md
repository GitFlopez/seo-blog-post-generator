# Prompt: Blog Outline + Title Variations

Use this before writing an article to build a complete content brief. Share with a writer or use it to run the full-article prompt.

---

## Prompt

You are an SEO content strategist. Create a detailed content brief and outline for the article described below.

**Brief:**
- Topic: [YOUR TOPIC, e.g., "project management software for freelancers"]
- Primary Keyword: [e.g., "project management software for freelancers"]
- Secondary Keywords: [e.g., "freelance project tracking, best PM tools, task management for freelancers"]
- Target Audience: [e.g., "freelancers 2-5 years in, juggling 3+ clients, frustrated with spreadsheets"]
- Competitors to Beat: [optional: paste 1-3 URLs of top-ranking articles]
- Target Word Count: [default: 1,500-2,000 words]

**Deliverables:**

### 1. Title Variations (3 options)
Provide exactly 3 title options:
- **Option A — Listicle:** "X Best [Topic] for [Audience] in [Year]" format
- **Option B — How-To:** "How to [Achieve Result] with [Topic]" format
- **Option C — Question:** A direct question format that matches a Google PAA query
For each: include character count (aim 50-60 chars) and note target keyword placement.

### 2. Content Brief
- **Search Intent:** Is this informational / commercial / navigational? What does the searcher want?
- **SERP Features to Target:** Featured snippet / listicle / PAA / video — which is most achievable and how to structure for it?
- **Content Angle:** What unique angle makes this article better than top-ranking results? (Not just longer — specifically *better for the reader*)

### 3. Full Outline
Structure: H1 > Introduction > H2 sections > Conclusion

For each H2 section include:
- The H2 header text (keyword-optimized)
- 2-3 H3 bullet points (specific sub-topics to cover)
- Target word count for that section
- Which keyword(s) to use in this section

Aim for 8-10 H2 sections total.

### 4. Keyword Placement Map
Table format:
| Keyword | Where to Use | Frequency |
|---------|-------------|-----------|
| [primary keyword] | H1, intro, 2-3 H2s, conclusion | 8-12× |
| [secondary 1] | ... | ... |

### 5. Internal Linking Opportunities
List 4-5 related topics this article should link to (even if those pages don't exist yet — flag them as "create this page").

Output the complete brief now. Keep it scannable — use headers, bullets, tables.
